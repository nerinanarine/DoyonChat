# P4-003: コードブロックの実行機能

## 概要

AI が生成した JavaScript/Python コードを、ブラウザ上または Sandpack で実行できるようにする。

## 受け入れ条件

1. コードブロックに「実行」ボタンが表示される（対応言語のみ）
2. 実行結果がコードブロック下に表示される
3. 実行はサンドボックス化され、ブラウザに影響を与えない
