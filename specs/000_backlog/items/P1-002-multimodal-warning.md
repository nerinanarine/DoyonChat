# P1-002: マルチモーダル非対応モデル警告

## 概要

OpenCode Go API でマルチモーダル対応なのは `glm-5.2` と `glm-5.1` のみ。それ以外のモデル（Kimi, DeepSeek, Qwen, MiniMax, MiMo, Hy3）で画像付きメッセージを送信すると、画像が無視される可能性がある。

## 受け入れ条件

1. 非マルチモーダルモデル選択中に画像をアップロードすると、警告バナーが表示される
2. マルチモーダルモデル選択中に画像をアップロードすると、警告は表示されない
3. 警告には「画像を削除」「このまま送信」の選択肢がある
