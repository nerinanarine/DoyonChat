# P3-003: 会話のエクスポート

## 概要

ユーザーが会話を JSON または Markdown 形式でダウンロードできるようにする。

## 受け入れ条件

1. 会話メニューから「エクスポート」オプションを選択できる
2. Markdown 形式でダウンロードされる（`# タイトル` + メッセージ一覧）
3. JSON 形式でもダウンロードできる（メタデータ含む）

## 関連ファイル

- `frontend/src/components/Sidebar/ConversationList.tsx`
- `frontend/src/services/chatApi.ts`
