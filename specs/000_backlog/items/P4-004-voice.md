# P4-004: 音声入力（Web Speech API）

## 概要

ブラウザの `SpeechRecognition` API を使用して、音声でメッセージを入力できるようにする。

## 受け入れ条件

1. マイクボタンをクリックして音声入力を開始できる
2. 音声認識結果がテキスト入力欄に反映される
3. 認識中は「聞き取り中…」のインジケーターを表示する
