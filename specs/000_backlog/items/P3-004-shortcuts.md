# P3-004: キーボードショートカット

## 概要

パワーユーザーの効率を上げるため、主要な操作にキーボードショートカットを割り当てる。

## 受け入れ条件

| ショートカット | 動作 |
|--------------|------|
| `Ctrl/Cmd + K` | 新規会話作成 |
| `Ctrl/Cmd + /` | 検索ボックスにフォーカス |
| `Ctrl/Cmd + Shift + ↑/↓` | 前/次の会話に切り替え |
| `Escape` | ストリーミング停止（生成中のみ） |

## 関連ファイル

- `frontend/src/App.tsx`
- `frontend/src/hooks/useChat.ts`
