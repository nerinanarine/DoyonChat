# P3-001: ダークモード

## 概要

MVP では除外されていた機能。Tailwind CSS の `dark:` 修飾子を使用してダークモードを実装する。

## 受け入れ条件

1. ヘッダーまたは設定メニューからライト/ダーク/システム設定を切り替えられる
2. 選択は `localStorage` に保存され、ページリロード後も復元される
3. すべてのコンポーネントでダークモード時のスタイルが適用される

## 関連ファイル

- `frontend/src/index.css`
- `frontend/src/components/Layout/AppLayout.tsx`
- 各コンポーネント
