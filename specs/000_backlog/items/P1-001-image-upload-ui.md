# P1-001: 画像入力（マルチモーダル）フロントエンド UI

## 概要

バックエンド (`chat.ts`) は既に `imageBase64` に対応しているが、フロントエンドに画像アップロード UI が存在しない。ドラッグ＆ドロップまたはファイルピッカーで画像を選択し、Base64 エンコードして送信できるようにする。

## 受け入れ条件

1. チャット入力欄に画像アップロードボタン（クリップアイコン）が表示される
2. 画像ファイルを選択するとプレビューが表示される
3. 画像付きメッセージを送信すると、API リクエストに `imageBase64` が含まれる
4. 5MB を超える画像はアップロード前に拒否される
